# Swema3000-import
A super-simple macro-enabled spreadsheet to import & tabulate LOGP-mode measurements from [SWEMA 3000 logger](https://swema.com/product-category/instruments/) connected to a SWA03 omnidirectional anemometer.

The macro can read .txt or .log files exported by [Swemair Termina 2](https://swema.com/product/swematerminal-2/) or [PuTTY](https://putty.org/).

Let me know if you need the program to import data from the instrument's other logger modes or other sensors.
