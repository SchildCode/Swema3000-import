# Swema3000-import
A super-simple macro-enabled spreadsheet to read in logger data from [Swema 3000 anemometers](https://swema.com/product-category/instruments/) in LOPG-mode.

The macro can read .txt or .log files exported by [Swemair Termina 2](https://swema.com/product/swematerminal-2/) or [PuTTY](https://putty.org/).

Let me know if you need the program to read data logged from the instrument's other modes.
