# Swema3000-import
A super-simple macro-enabled spreadsheet to read in logger data from the [Swema 3000 anemometer](https://swema.com/product-category/instruments/) in LOPG-mode.
The macro cam read .txt or .log files exported by [Swemair Termina 2](https://swema.com/product/swematerminal-2/) or [PuTTY](https://putty.org/).
Let me know if you need the program to read in from other modes.
